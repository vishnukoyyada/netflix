Introduction to Computer Networks

Network Concepts

A computer network is a collection of interconnected computers and computing devices that can communicate and share resources with each other. Networks can be local (LAN) or wide-area (WAN).

  • Network Devices: Routers, Switches, Firewalls, etc.
  • Network Protocols: TCP/IP, HTTP, DNS, etc.
  • Network Topologies: Bus, Star, Mesh, etc.
  • Network Security: Firewalls, Encryption, Access Control, etc.

The Internet

The internet is a global system of interconnected computer networks that facilitates data communication between devices across the world.
